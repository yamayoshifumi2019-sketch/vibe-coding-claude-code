# English Vocabulary Learning App
Let's learn English by English ! This is the fastest way to master it !

## About

An English-only vocabulary learning app for intermediate learners (TOEFL/IELTS prep). Learn English through English - no translations!

## Features

- 260+ vocabulary words with definitions, examples, and usage tips
- Flashcard mode for studying
- Quiz mode (multiple choice)
- Add your own custom words
- Favorites and progress tracking
- Mobile-first responsive design

## Tech Stack

- Next.js 14 (App Router)
- Prisma + SQLite
- Tailwind CSS
- TypeScript

## Getting Started

```bash
npm install
npm run db:seed   # Load vocabulary data
npm run dev       # Start development server
```

Open [http://localhost:3000](http://localhost:3000) with your browser.
